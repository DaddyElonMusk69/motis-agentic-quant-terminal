## V0.3 Ensemble Voting Strategy (Final)

Implemented feature-audit-driven v0.3:
1. Three-timeframe ensemble (micro/local/macro) — equal votes
2. daily_range_position as tiebreaker only, never overrides majority
3. Cross-timeframe conflict: macro vs local oppose → SKIP
4. Confidence scales with vote agreement (strong/narrow/with-range-support)

| Monthly spread | 34.6% | 26.8% | -7.8pp |
| Worst month | 37.7% | 45.6% | +7.9pp |

### Key insight:
Conflict SKIP contributes +8.79pp alone. Without it, ensemble voting is 50.44%.
The conflict SKIP identifies ambiguous signals (macro vs local oppose = 48.6% agreement in v0.2) and correctly removes them.

## SOL Vegas EMA 5m Cluster — Stage 1A iter_002 v0.2→v0.3 (Directional Override)

### Context
- iter_002 training replay: v0.2 = 52.9% agreement (808/1528, 666 mismatch, 54 neutral)
- Failure cluster: mixed_mismatch_and_neutral
- 10 protected cases specified

### Root Cause Analysis
Feature audit revealed that when the 1d macro direction disagrees with the 2-1 majority vote, the strategy is consistently wrong. Worst pattern: micro=SHORT, local=SHORT, macro=LONG → 75% mismatch (baseline picks SHORT, truth is LONG).

### V0.3 Changes (1 rule, 1 threshold)
1. **Macro override**: When 1d return direction disagrees with majority (2-1 split) AND |1d return| ≥ 1.5%, → trust 1d macro over majority. Prevents the strategy from voting against a strong daily trend.
2. **Single-direction ENTER**: When only macro exists (1 direction) → ENTER with macro direction instead of SKIP.
3. **Tie-breaker priority**: Macro takes priority over daily_range_position.
4. Version bumped: v0.2 → v0.3.

### Justification
- Targets failure pattern `macro≠majority in 2-1 split` — 75% mismatch rate under v0.2
- Threshold (1.5%) prevents weak macro signals from overriding (avoided the 1 protected regression from unconditional macro override)
- Explainable: "Strong daily trends should not be overridden by shorter-term noise"
- Stage 1A direction-only — no entry gates, no trade management

### Results
| Metric | v0.2 | v0.3 | Δ |
|---|---|---|---|
| Match | 808 | 858 | +50 |
| Mismatch | 666 | 639 | -27 |
| Neutral | 54 | 31 | -23 |
| Agreement | 52.9% | 56.2% | +3.3pp |
| Protected | 0 regressed | 0 regressed | ✓ |
| Worst month | 36.1% (Apr26) | 42.0% (Apr26) | +5.9pp |

### Monthly: All months ≥ 42.0%, σ=9.2%, range 42.0-72.8%
